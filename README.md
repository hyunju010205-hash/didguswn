# 🌙 LUMÉ | Eternal Light  

> *Will you still love me when I'm no longer young and beautiful...*  

LUMÉ는 **"빛(Eternal Light)"**을 모티프로 한 하이엔드 패션 & 뷰티 브랜드 웹사이트입니다.  
이 프로젝트는 감성적인 럭셔리 브랜드 경험을 **HTML + CSS + JavaScript**로 정제된 구조와 미니멀한 디자인을 통해 구현했습니다.

---

## 🖋️ 프로젝트 소개

**LUMÉ | Eternal Light**은 "시간이 흘러도 변치 않는 아름다움"이라는 컨셉 아래 제작된 웹사이트입니다.  
정적인 이미지가 아닌, 감성적인 브랜드 무드와 부드러운 인터랙션을 통해  
**명품 브랜드의 품격과 감정적 몰입감**을 전달합니다.

---

## ✨ 주요 특징

- **Full-screen Hero Section**  
  : 브랜드 시그니처 문구와 영상(또는 이미지) 중심의 메인 비주얼  

- **Smooth Scrolling Navigation**  
  : 자연스럽게 연결되는 페이지 전환  

- **Responsive Layout**  
  : 데스크톱, 모바일 모두 대응 가능한 반응형 구조  

- **Collection Gallery**  
  : 제품 컬렉션(패션, 뷰티)을 감각적으로 전시  

- **About & Contact Section**  
  : 브랜드 철학과 뉴스레터 구독 기능 포함
