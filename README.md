LUMÉ | Eternal Light

Will you still love me when I'm no longer young and beautiful...

---

프로젝트 개요  
LUMÉ는 럭셔리 감성과 미니멀리즘을 결합한 패션·뷰티 브랜드 웹사이트입니다.  
심플한 구조와 감각적인 디자인으로, 브랜드의 정체성과 무드를 시각적으로 표현했습니다.

주요 특징  
- 풀스크린 이미지와 감성적인 타이포그래피  
- 부드러운 스크롤 전환 효과  
- 반응형 디자인 (모바일, 데스크탑 모두 대응)

---

설치 및 실행 방법  
1. 저장소 클론  
   ```bash
   git clone https://github.com/hyunju010205/lume-eternal-light.git
````

2. 프로젝트 폴더로 이동

   ```bash
   cd lume-eternal-light
   ```
3. index.html 파일을 브라우저에서 실행

---

폴더 구조

```bash
lume-eternal-light/
│
├── index.html
├── /assets
├── /css
│   └── style.css
└── /js
    └── main.js
```

---

주요 기능

* 홈 화면: 브랜드 문구와 감성 배경 이미지
* 컬렉션: 시즌별 룩북 및 제품 소개
* 어바웃: 브랜드 철학 및 정체성
* 컨택트: 뉴스레터 구독 및 문의

---

기술 스택
HTML5 / CSS3 / JavaScript
Google Fonts (Playfair Display, Inter)
반응형 디자인 및 스크롤 애니메이션 적용
